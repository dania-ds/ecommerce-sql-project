# E-Commerce SQL Project

This project contains a simple E-Commerce database I built for practicing SQL. It includes:

* Database tables (Customers, Products, Orders, Order_Items)
* Sample test data
* A set of SQL queries focused on understanding joins, grouping, and basic analysis

The goal of this project was to practice:

* Creating tables
* Inserting data
* Using SELECT, JOIN, GROUP BY, and simple aggregations

## Project Structure

```
schema/         Tables creation SQL file
data/           Insert statements
queries/        SQL queries I practiced
screenshots/    Output screenshots
```

## How to Run

1. Run the table creation file
2. Insert the sample dataset
3. Execute the queries to view results

## Purpose

This project was created for learning and practicing SQL basics using a small, easy E-Commerce dataset.
